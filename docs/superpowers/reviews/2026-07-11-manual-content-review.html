Critical13 pages bypass the component kit

Spec §"Semantic Component Kit" designed <Recipe>, <Steps>, <MaterialList> specifically so recipe pages render as styled, semantic markup. These 13 pages instead paste the PDF's multi-column layout verbatim into <pre className="overflow-x-auto whitespace-pre"> blocks — words split across columns, empty bullets, "Procedimiento"/"Rendimiento"/"Tiempo" hardcoded as plain text instead of composed via <Recipe>/<Steps>.
